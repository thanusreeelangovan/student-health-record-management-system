# Student Health Record Update

## Project Description
The **Student Health Record Update** project helps schools keep track of students’ health information. It allows teachers or administrators to add, view, and update health records for each student, including their class, section, and medical details. The system makes managing student health data organized and easy to use.

## Features
- Add new student health records  
- Update existing records  
- Search and filter records by student name, class, or section  
- User-friendly graphical interface (GUI)  

## Tools Used
- Python – programming language  
- Tkinter – for GUI  
- MySQL – for storing data
